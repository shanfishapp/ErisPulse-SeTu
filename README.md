# ErisPulse-SeTu
这是一个ErisPulse的色图插件，API地址：<https://api.lolicon.app/setu/v2>  
您可以使用其中之一的指令来触发本插件：
- `/随机色图`
- `随机色图`
- `/色图`，
- `色图`


在某些情况下，部分功能可能失效：
- 1.平台不支持编辑消息时，无法实现消息更改内容，原始消息提示内容将**不会修改**。
- 2.平台不支持发送图片时，将**无法运行本模块**。虽然可以正常初始化模块，但并无实际作用。
